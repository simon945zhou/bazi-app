# 天纪算命 App - 安装指南

## 📱 手机安装说明

您的网页 app 现在已经支持安装到手机主屏幕，像原生 app 一样使用！

### Android 手机安装步骤：

1. **打开应用**：在手机浏览器（Chrome/Edge）中访问 `http://您的服务器地址:5000`
2. **添加到主屏幕**：
   - 点击浏览器右上角的 ⋮（菜单）
   - 选择"添加到主屏幕"或"安装应用"
   - 点击"添加"或"安装"
3. **使用 App**：在主屏幕找到"天纪算命"图标，点击即可像原生 app 一样使用

### iPhone (iOS) 安装步骤：

1. **打开应用**：在 Safari 浏览器中访问 `http://您的服务器地址:5000`
2. **添加到主屏幕**：
   - 点击底部的"分享"按钮 📤
   - 向下滚动，选择"添加到主屏幕"
   - 点击右上角的"添加"
3. **使用 App**：在主屏幕找到"天纪算命"图标，点击即可使用

## 🌟 PWA 特性

安装后享受以下功能：

- ✅ **离线访问**：无网络也能打开应用界面
- ✅ **全屏体验**：隐藏浏览器地址栏，更像原生 app
- ✅ **快速启动**：直接从主屏幕一键打开
- ✅ **自动更新**：后台静默更新到最新版本

## 🚀 部署到公网（可选）

如果想让手机随时随地访问，需要将应用部署到公网服务器：

### 方案 1：使用内网穿透工具（推荐初学者）
- 使用 ngrok, localtunnel, 或花生壳等工具
- 将 `http://127.0.0.1:5000` 映射到公网域名

### 方案 2：云服务器部署
1. 购买云服务器（阿里云、腾讯云等）
2. 上传项目文件
3. 安装依赖：`pip install -r requirements.txt`
4. 使用 Gunicorn + Nginx 部署
5. 配置 HTTPS（PWA 在 HTTPS 下体验更佳）

## ⚠️ 当前限制

- **网络依赖**：八字计算需要连接后端服务器，完全离线时无法进行新的推算
- **本地访问**：目前只能在同一 WiFi 网络下访问（除非部署到公网）

## 📦 已生成的文件

- `/static/manifest.json` - PWA 配置文件
- `/static/sw.js` - Service Worker（离线缓存）
- `/static/images/icon-192.png` - 应用图标（小）
- `/static/images/icon-512.png` - 应用图标（大）
