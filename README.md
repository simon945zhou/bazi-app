# 天纪算命系统

这是一个基于倪海厦老师《天纪》内容开发的算命系统，包含个人算命、配对分析和知识库管理功能。

## 功能特点

1. **个人算命**：根据输入的生辰八字和性别，生成详细的命理分析报告
2. **配对分析**：分析两人的八字配对情况，提供合婚建议
3. **知识库管理**：包含《天纪》相关知识，可以添加、删除和查看知识内容

## 安装说明

### 前提条件
- 已安装Python 3.6或更高版本

### 安装依赖

1. 打开命令行终端
2. 进入项目目录
3. 运行以下命令安装所需依赖：

```bash
pip install -r requirements.txt
```

## 使用方法

### 启动应用程序

在命令行终端中，进入项目目录，运行以下命令：

```bash
python main_app.py
```

### 个人算命
1. 在"个人算命"标签页中，选择出生日期和时间
2. 选择性别
3. 点击"开始算命"按钮
4. 查看生成的命理分析报告

### 配对分析
1. 在"配对分析"标签页中，分别输入两个人的出生日期、时间和性别
2. 点击"分析配对"按钮
3. 查看配对分析结果

### 知识库管理
1. 在"知识库管理"标签页中，可以查看、添加和删除知识内容
2. 点击"添加知识"按钮可以添加新的知识
3. 选择知识项后点击"删除知识"按钮可以删除选中的知识
4. 点击"刷新列表"按钮可以刷新知识列表

## 注意事项

1. 本系统仅供学习和参考，请勿完全依赖其结果
2. 八字计算和分析基于传统命理理论，可能存在局限性
3. 知识库中的内容会保存在本地，不会上传到任何服务器
