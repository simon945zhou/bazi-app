### ❌ Node.js - 未安装
**当前任务：立即安装**

**下载地址：**
https://nodejs.org/

**安装步骤：**
1. 点击上方链接，下载 "LTS" 版本（推荐，左边的绿色按钮）
2. 下载完成后，双击 `.msi` 安装包
3. 安装向导：
   - [√] Accept the terms
   - [√] 默认安装路径（C:\Program Files\nodejs\）
   - [√] **重要：勾选 "Automatically install the necessary tools"**
   - [√] 点击 Install
4. 等待安装完成（约 3-5 分钟）
5. **关闭当前所有命令行窗口**
6. **打开新的 PowerShell 窗口**，运行以下命令验证：
   ```
   node -v
   npm -v
   ```
7. 如果显示版本号（如 v20.11.0 和 10.2.4），说明安装成功 ✅
